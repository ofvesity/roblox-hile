# 🌌 Vesity Hub

**Vesity Hub**, Roblox platformu için en gelişmiş, optimize edilmiş ve kullanıcı dostu script topluluğudur. Oyun deneyiminizi bir üst seviyeye taşımak için performans ve güvenlik odaklı kodlanmıştır.

---

## 🚀 Neden Vesity Hub?

Sıradan scriptlerin aksine, Vesity Hub doğrudan bellek optimizasyonu ve düşük gecikme (MS) odaklı çalışır. Cihazınızı yormaz, FPS düşüşü yaşatmaz.

* ⚡ **Ultra Hızlı Execution:** Gecikmesiz `loadstring` mimarisi.
* 🛡️ **Anti-Ban Koruması:** Sürekli güncellenen bypass modülleri.
* 🎨 **Vesity UI:** Modern, karanlık tema destekli ve tamamen özelleştirilebilir arayüz.
* 📱 **Cross-Platform:** Hem PC hem de mobil cihazlarda tam uyumluluk.
* 🔄 **Cloud-Based:** Script güncellemeleri otomatik olarak yüklenir, tekrar kopyalamanıza gerek kalmaz.

---

## 📥 Kurulum ve Çalıştırma

Vesity Hub'ı kullanmak oldukça basittir. Aşağıdaki adımları takip edin:

1. Aşağıdaki **Script'i Kopyalayın**.
2. Kullandığınız **Executor'ı** (Çalıştırıcı) açın.
3. Roblox'a **Attach** edin ve kodu yapıştırıp **Execute** butonuna basın.

### 📋 Ana Script (Loadstring)

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/ofvesity/roblox-hile/main/vesity.lua"))()
```
